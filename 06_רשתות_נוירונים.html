שיעור 9 · נושא 06

רשתות נוירונים

AIMA 18.7 · פרספטרון · סיגמואיד · Forward Pass · Backpropagation · LR=0.9

שאלת הרשת במבחן תמיד באותו פורמט: נתונה רשת קטנה עם משקולות, הטיות (bias), קלט ומטרה. צריך (1) לחשב את הפלט במעבר קדימה (Forward), ואז (2) לעדכן את המשקולות לפי Backpropagation עם קצב למידה LR. נעבור על הדוגמה הקנונית של AIMA (רשת 3→2→1) מתחילתה ועד סופה — בדיוק כפי שמופיעה בשקפים.

1. מודל הנוירון

קלט נטו
I_j = Σ wᵢⱼ·Oᵢ + θⱼ — סכום משוקלל של הקלטים פלוס ההטיה θⱼ של הנוירון.
פונקציית הפעלה
סיגמואיד: O_j = σ(I_j) = 1 / (1 + e−I_j) — מועכת לטווח (0,1).
נגזרת הסיגמואיד
σ'(I) = O·(1 − O) — קריטית ל-Backprop.

📐 ערכי סיגמואיד שימושיים

σ(0)=0.5 · σ(−0.7)≈0.332 · σ(0.1)≈0.525 · σ(−0.105)≈0.474 · σ(1)≈0.731. (במחשבון: חשב e−I ואז 1/(1+...)).

2. הרשת שנעבוד עליה (3→2→1)

שלושה קלטים (צמתים 1,2,3), שתי יחידות נסתרות (4,5), ויחידת פלט (6). קלט x₁=1, x₂=0, x₃=1, מטרה T=1, LR=0.9, סיגמואיד.

הרשת: לכל יחידה נסתרת/פלט יש גם הטיה θ. משקל wᵢⱼ = מצומת i לצומת j.

3. מעבר קדימה (Forward Pass)

  1. יחידות נסתרות (4, 5):
    I₄ = 0.2·1 + 0.4·0 + (−0.5)·1 + (−0.4) = −0.7 ⟹ O₄ = σ(−0.7) = 0.332
    I₅ = (−0.3)·1 + 0.1·0 + 0.2·1 + 0.2 = 0.1 ⟹ O₅ = σ(0.1) = 0.525
  2. יחידת הפלט (6):
    I₆ = (−0.3)·0.332 + (−0.2)·0.525 + 0.1 = −0.105
    O₆ = σ(−0.105) = 0.474

הפלט של הרשת: O₆ = 0.474 (המטרה היא 1, אז יש שגיאה לתקן).

4. Backpropagation — עדכון משקולות (LR=0.9)

📐 הנוסחאות

Err (פלט) = O·(1−O)·(T−O)
Err (נסתר) = O·(1−O)·Σ(Errk·wjk)
w ← w + LR·Errj·Oi  ·  θ ← θ + LR·Errj
  1. שגיאה ביחידת הפלט: Err₆ = O₆·(1−O₆)·(T−O₆) = 0.474·0.526·0.526 = 0.1311
  2. שגיאות ביחידות הנסתרות (משתמשים ב-w הישנים!):
    Err₄ = O₄·(1−O₄)·Err₆·w₄₆ = 0.332·0.668·0.1311·(−0.3) = −0.0087
    Err₅ = O₅·(1−O₅)·Err₆·w₅₆ = 0.525·0.475·0.1311·(−0.2) = −0.0065
  3. עדכון משקולות הפלט (הקלט אליהן הוא O₄,O₅):
    w₄₆ ← −0.3 + 0.9·0.1311·0.332 = −0.261
    w₅₆ ← −0.2 + 0.9·0.1311·0.525 = −0.138
  4. עדכון משקולות הקלט (הקלט אליהן הוא x₁,x₂,x₃):
    w₁₄ ← 0.2 + 0.9·(−0.0087)·1 = 0.192 · w₃₄ ← −0.5 + 0.9·(−0.0087)·1 = −0.508
    w₁₅ ← −0.3 + 0.9·(−0.0065)·1 = −0.306 · w₃₅ ← 0.2 + 0.9·(−0.0065)·1 = 0.194
    w₂₄ ← 0.4 (ללא שינוי) · w₂₅ ← 0.1 (ללא שינוי) — כי x₂=0.
  5. עדכון הטיות: θ₆ ← 0.1 + 0.9·0.1311 = 0.218 · θ₄ ← −0.408 · θ₅ ← 0.194

🎯 שימו לב

משקל שהקלט אליו הוא 0 (כמו w₂₄, w₂₅ כש-x₂=0) לא משתנה, כי העדכון מוכפל בקלט. זו בדיקת שפיות מצוינת. (ההטיה, לעומת זאת, תמיד מתעדכנת — כי "הקלט" שלה הוא 1.)

5. שיטת פתרון — "חשב פלט ועדכן משקולות"

  1. Forward — חשב I (כולל הטיה) ו-O=σ(I) לכל יחידה, שכבה אחר שכבה (מהקלט לפלט).
  2. שגיאת הפלט — Err₆ = O₆(1−O₆)(T−O₆).
  3. שגיאות נסתרות — Errⱼ = Oⱼ(1−Oⱼ)·Σ(Errk·wjk) (עם המשקולות הישנים!).
  4. עדכן משקולות והטיות — w ← w+LR·Err·O, θ ← θ+LR·Err. עגל ל-3–4 ספרות.

⚠️ טעויות נפוצות

  • לשכוח להוסיף את ההטיה θ בחישוב I.
  • להשתמש במשקולות הפלט המעודכנים בחישוב Err של השכבה הנסתרת — חובה להשתמש בישנים.
  • לשכוח את הגורם O(1−O) (נגזרת הסיגמואיד) ב-Err.
  • להכפיל ב-O/x הלא נכון — המשקל מתעדכן לפי הקלט ש"נכנס" אליו.
  • לבלבל סימן: כשהמטרה גדולה מהפלט, (T−O)>0 והמשקולות גדלים.

📐 לסיכום הנושא

  • Forward: I=Σwo+θ, O=σ(I).
  • Err_פלט=O(1−O)(T−O) · Err_נסתר=O(1−O)ΣErr·w.
  • עדכון: Δw=LR·Err·input, Δθ=LR·Err; קלט 0 → אין שינוי במשקל.

🧭 ההיגיון שמאחורי ההסתעפויות

זו זרימה ליניארית בשני חצאים: קודם Forward מלא (מחשבים את כל ה-O), ואז Backprop לאחור — שגיאה בפלט → שגיאות בנסתר (עם המשקולות הישנים!) → עדכון כל המשקולות וההטיות. שתי תיבות האזהרה מסמנות את שתי הטעויות הקריטיות ביותר.
