שיעור 9 · נושא 06

רשתות נוירונים

AIMA 18.7 · פרספטרון · סיגמואיד · Forward Pass · Backpropagation · η=0.3

השאלה במבחן (2019, 2023) תמיד באותו פורמט: נתונה רשת קטנה עם משקולות, קלט ומטרה. צריך (1) לחשב את הפלט במעבר קדימה (Forward), ואז (2) לעדכן את המשקולות לפי Backpropagation עם קצב למידה η=0.3. נעבור על דוגמה מספרית מלאה.

1. מודל הנוירון

קלט נטו
net = Σ wᵢ·xᵢ (+ הטיה bias אם יש) — סכום משוקלל של הקלטים.
פונקציית הפעלה
סיגמואיד: σ(net) = 1 / (1 + e−net) — מועכת לטווח (0,1).
נגזרת הסיגמואיד
σ'(net) = σ(net)·(1 − σ(net)) = o·(1−o) — קריטית ל-Backprop.

📐 ערכי סיגמואיד שימושיים

σ(0)=0.5 · σ(0.5)≈0.6225 · σ(0.3)≈0.5744 · σ(1)≈0.731 · σ(2)≈0.881. (במחשבון: חשב e−net ואז 1/(1+...)).

2. הרשת שנעבוד עליה

רשת 2-2-1: שני קלטים, שני נוירונים נסתרים, פלט אחד. קלט x₁=1, x₂=0, מטרה t=1, η=0.3, סיגמואיד.

הרשת: w משקולות קלט→נסתר, v משקולות נסתר→פלט.

3. מעבר קדימה (Forward Pass)

  1. נוירונים נסתרים:
    net_{h1}=0.5·1+0.1·0=0.5 ⟹ h₁=σ(0.5)=0.6225
    net_{h2}=0.3·1+0.8·0=0.3 ⟹ h₂=σ(0.3)=0.5744
  2. נוירון הפלט:
    net_o=0.2·0.6225+0.9·0.5744=0.1245+0.5170=0.6415
    o=σ(0.6415)=0.6551

הפלט של הרשת: o = 0.6551 (המטרה היא 1, אז יש שגיאה לתקן).

4. Backpropagation — עדכון משקולות (η=0.3)

📐 הנוסחאות

δ_o = (t − o)·o·(1−o)
δ_h = h·(1−h)·(δ_o·v)
משקל ← משקל + η · δ · (קלט לאותו משקל)
  1. דלתא בפלט: δ_o=(1−0.6551)·0.6551·(1−0.6551)=0.3449·0.6551·0.3449=0.0779
  2. עדכון משקולות הפלט (הקלט אליהן הוא h₁,h₂):
    v₁ ← 0.2+0.3·0.0779·0.6225 = 0.2+0.0146 = 0.2146
    v₂ ← 0.9+0.3·0.0779·0.5744 = 0.9+0.0134 = 0.9134
  3. דלתא בשכבה הנסתרת (משתמשים ב-v הישנים):
    δ_{h1}=0.6225·0.3775·(0.0779·0.2)=0.2350·0.01558=0.0037
    δ_{h2}=0.5744·0.4256·(0.0779·0.9)=0.2445·0.07011=0.0172
  4. עדכון משקולות הקלט (הקלט אליהן הוא x₁,x₂):
    w₁₁ ← 0.5+0.3·0.0037·1 = 0.5011
    w₁₂ ← 0.3+0.3·0.0172·1 = 0.3051
    w₂₁ ← 0.1 (ללא שינוי, x₂=0) · w₂₂ ← 0.8 (ללא שינוי)

🎯 שימו לב

משקל שהקלט אליו הוא 0 (כמו w₂₁, w₂₂ כש-x₂=0) לא משתנה, כי העדכון מוכפל בקלט. זו בדיקת שפיות מצוינת.

5. שיטת פתרון — "חשב פלט ועדכן משקולות"

  1. Forward — חשב net ו-o לכל נוירון, שכבה אחר שכבה (מהקלט לפלט).
  2. δ בפלט — δ_o=(t−o)·o·(1−o).
  3. עדכן משקולות הפלט — v ← v+η·δ_o·h.
  4. δ בנסתר — δ_h=h(1−h)·Σ(δ_o·v) (עם המשקולות הישנים!).
  5. עדכן משקולות הקלט — w ← w+η·δ_h·x. עגל ל-4 ספרות.

⚠️ טעויות נפוצות

  • להשתמש במשקולות הפלט המעודכנים בחישוב δ_h — חובה להשתמש בישנים.
  • לשכוח את הגורם o(1−o) (נגזרת הסיגמואיד) ב-δ.
  • להכפיל ב-x הלא נכון — המשקל מתעדכן לפי הקלט ש"נכנס" אליו.
  • לבלבל סימן: כשהמטרה גדולה מהפלט, (t−o)>0 והמשקולות גדלים.
  • לא לעגל באופן עקבי (השאלה מבקשת 4 ספרות עשרוניות).

📐 לסיכום הנושא

  • Forward: net=Σwx, o=σ(net).
  • δ_פלט=(t−o)o(1−o) · δ_נסתר=o(1−o)Σδv.
  • עדכון: Δw=η·δ·input; קלט 0 → אין שינוי.

🧭 ההיגיון שמאחורי ההסתעפויות

זו זרימה ליניארית בשני חצאים: קודם Forward מלא (מחשבים את כל ה-o), ואז Backprop לאחור — דלתא בפלט → עדכון משקולות פלט → דלתא בנסתר (עם המשקולות הישנים!) → עדכון משקולות קלט. שתי תיבות האזהרה מסמנות את שתי הטעויות הקריטיות ביותר.
